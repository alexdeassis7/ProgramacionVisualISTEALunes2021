Etiquetas HTML

Este ejemplo muestra como combinar algunas de las etiquetas mas basicas de HTML5 .
Recuerda que es importante la diferencia entre ella

Etiqueta UL + LI

Si listamos personas nominadas a los Oscars , dado que el orden no altera el significado , debemos usar ul.

  • David Verdaguer
  • Bratt Pit
  • Daniel Rovira
  • Israel Elejalde

Etiqueta OL + LI

En el caso de que estemos listando elementos donde el orden es importante, como por ejemplo la clasificacion de un mundial de fitbol , debemos usar ol.

  1. Grecia
  2. Alemania
  3. Argentina
  4. Uruguay

Listas no Ordenadas

  • Esto es un tipo de punto
  • Este es otro
  • Y este es otro diferente

Listas Ordenadas

  1. Este sera el numero 20.
  2. Este sera el 21
  3. Este sera el 22. Y asi sucesivamente...

Listas De Sefiniciones
